Project #1: Fireworks!

My Project was creating a functioning firework! We made it so that using the bottom rows of keys on your keyboard could create a firework that goes off at a certain height with a variable of colors and shapes.

Project #2: Pumpkin Drop!

The point of this project was to create a game where you must memorize the numbers of each pumpkin before they are hidden, then click keys 1-4 to drop them while using the basket to catch them.

Project #3: Veterans Day Card!

We created a Happy Veterans Day card featuring fireworks, background noises & music, and background images.

Project #4: Fish Tank Monitor!

We created a fish tank monitor that we secured from possible cyber attacks.

Project #5: Dance Game!

We created a game where pressing the corresponding arrows that are shown on the screen, on your keyboard, make your character dance.

Project #6: Rover Data!

We had to figure out the location of the rover based on light, temperature, light and gyroscope.
